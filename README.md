# Коттеджи Сарженка — Интерактивная визитка

Лёгкая адаптивная HTML-визитка без внешних зависимостей.

## Структура

```
index.html        — единственная страница (15 KB)
assets/
  video.mp4       — видео в карточке
  shashlik.png    — резервное изображение
```

## Особенности

- Нет внешних зависимостей: ни CDN, ни npm-пакетов
- Тёмно-зелёный фон с золотыми акцентами
- Анимированный фон: плавающие размытые пятна + падающие листья (чистый CSS)
- Видео в карточке: autoplay / muted / loop / playsinline
- Адаптивная вёрстка для мобильных и десктоп

## Публикация на GitHub Pages

Settings → Pages → Source: `main` branch, `/ (root)`
