# Визитка Коттеджи Сарженка

Лёгкая адаптивная HTML-визитка без внешних зависимостей, готовая к публикации на GitHub Pages.

## Стек

- Чистый HTML/CSS/JS — никаких фреймворков, CDN, npm-пакетов
- Один файл `index.html` (15 KB) + папка `assets/`

## Структура

```
index.html          — единственная страница сайта
assets/
  video.mp4         — видео в центральной карточке
  shashlik.png      — резервное фото (не используется на странице)
```

## Дизайн

- Фон: `#0b2219` (тёмно-зелёный)
- Акценты: `#d4af37` (золото)
- Текст: `#f5f5dc` (кремовый)
- Шрифты: системные — `Georgia, serif` и `system-ui, sans-serif`
- Стеклянная карточка: `backdrop-filter: blur(14px)` + золотое свечение

## Анимации (чистый CSS)

- 4 размытых орба дрейфуют по фону (28–40 сек цикл)
- 12 падающих листочков-SVG с разными задержками (15–25 сек цикл)
- Карточка: плавное появление `fadeInUp` при загрузке
- Кнопка карты: лёгкая пульсация
